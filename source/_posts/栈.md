# 栈

## 栈的定义：
+ 栈（stack)是一种元素满足后进先出（Last in first out,LIFO) 规则的线性表；
+ 对栈的操作是在 表尾 进行；
+ ** 栈底 （bottom)**：栈的表头，** 栈顶 (top)**：栈的表尾；
+ ** 入栈 (push)**：向栈内添加元素，** 出栈 (pop)**：删除元素；
+ ** 空栈 **：没有任何元素的栈；
+ 栈的主要操作：入栈、出栈、判断栈空；

public:
    Stack(int size)
    {
        maxSize = size;                     //设置最大容量

## c++ stack的使用：
+ 头文件：
```cpp
#include <stack>
```
+ 声明一个栈：
```cpp
std::stack <DataType> p;
```
+ 相关函数：
```cpp
p.empty();		//如果是空栈，返回真值，否则返回假值
p.size();		//返回栈中元素的个数
p.top();		//返回栈顶元素的**值**，但不删除该元素
p.pop();		//删除栈顶元素，但不返回其值
p.push(k);		//压入新元素k
```
